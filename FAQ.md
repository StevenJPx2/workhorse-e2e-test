# Frequently Asked Questions

## What is this repository for?

This repository provides documentation and resources for this project.
For a full overview of its purpose and contents, see [README.md](README.md).

## How are changes verified?

There is no build system or automated test suite. Changes are verified by
reviewing the output of `git diff` before committing. See [README.md](README.md)
for more detail on the project's conventions.

## How do I get help?

For support options and how to reach the maintainers, see [SUPPORT.md](SUPPORT.md).
