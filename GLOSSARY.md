# Glossary

Terms specific to this repository and the Workhorse project.

| Term | Definition | Source |
|---|---|---|
| **Workhorse** | The overarching system or service under test, whose end-to-end behaviour this repository exercises. | README.md |
| **E2E runs** (End-to-End runs) | Automated test executions that exercise a complete user- or system-level flow from start to finish, as opposed to isolated unit tests. | README.md |
| **Test target** | The subject repository or artifact that the E2E suite is pointed at and exercises during a run. | README.md |
