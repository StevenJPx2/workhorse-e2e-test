# Glossary

Definitions for Workhorse-specific terms used in this repository.

| Term | Definition | Source |
|------|------------|--------|
| **Workhorse** | The core system or application under test whose end-to-end behaviour this repository validates. | `README.md` |
| **E2E runs** | End-to-end test executions that exercise the full Workhorse system from input to output without mocking internal components. | `README.md` |
| **Test target** | The specific build artifact or deployment of Workhorse that E2E runs are executed against. | `README.md` |
