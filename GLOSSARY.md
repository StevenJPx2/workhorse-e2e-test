# Glossary

Workhorse-specific terms used across the documentation in this repository.

---

**Workhorse** — The overarching system or platform for which this repository provides end-to-end test infrastructure. *(Appears in: README.md)*

**E2E (End-to-End)** — A testing methodology that validates a complete workflow from start to finish, exercising all integrated components together. *(Appears in: README.md)*

**E2E run** — A single execution of the full end-to-end test suite against a Workhorse environment. *(Appears in: README.md)*

**Test target** — The specific Workhorse deployment or component under test during an E2E run. *(Appears in: README.md)*

**workhorse-e2e-test** — The name of this repository; the dedicated test project that houses E2E tests for the Workhorse platform. *(Appears in: README.md, SUPPORT.md)*
